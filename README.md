
## Sobre a atividade

Esta atividade foi desenvolvido para colocar em prática conceitos do Tailwind, com o foco principal foi criar uma interface limpa e responsiva.

### Principais funcionalidades:

*  Design totalmente responsivo (Mobile First).
*  customização com Tailwind CSS.
*  código semântico.

## Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes ferramentas:
* **HTML**: Estruturação semântica do conteúdo.
* **Tailwind**: Framework utilizado.
